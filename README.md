## How to run tutorials

### System Requirements
* jdk version 11 or later
* docker-compose
* node.js version 14 or later
* npm version 8 or later

### Build tutorial sample application
```
sh build_and_run_totorial.sh
```

### See tutorial
http://localhost:7007


## Run test (jest)
```
cd tutorial+test
npm install
npm run test
```
